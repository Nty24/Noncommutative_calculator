
Noncommutative calculator is an application that simplifies either commutative or noncommutative mathematical expressions. 
Simply, type a mathematical expression and click on the button "simplify" to obtain a simplified expression together with its LaTeX code.

This app might be useful for algebraists, operator algebraists and other mathematicians. 


Caution: This application was built in Python, and because of this you may use either '**' or '^' as a power.

I hope that you find it useful.
